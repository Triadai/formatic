Loader plugin (loader)

When metadata isn't available, we ask the loader to load it. The loader will try to find an appropriate source based on the metadata keys.

Note that we ask the loader to load metadata with a set of keys like ['foo', 'bar'], but those are converted to a single key like foo::bar for the sake of caching.
