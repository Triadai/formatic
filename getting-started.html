Getting started

Installing and using Formatic

Setup

You can import Formatic as a CommonJS module or as a global.

CommonJS

$ npm install formatic

// Get the Formatic class.
var Formatic = require('formatic');

// Create some fields.
var fields = [
  {
    type: 'single-line-string',
    key: 'firstName',
    label: 'First Name'
  },
  {
    type: 'single-line-string',
    key: 'lastName',
    label: 'Last Name'
  }
];

// Render the form.
React.render(React.createElement(Formatic, {fields: fields}), document.body);

Global

$ bower install formatic

Basic Usage

Basic usage of Formatic is pretty simple. Formatic is just a React class. Pass in the fields as props to render your fields. Again, the simple example from above:

// Create some fields.
var fields = [
  {
    type: 'single-line-string',
    key: 'firstName',
    label: 'First Name'
  },
  {
    type: 'single-line-string',
    key: 'lastName',
    label: 'Last Name'
  }
];

// Render the form.
React.render(
  <Formatic fields={fields}/>
, document.body);

The live example is the same. The only difference is that we render to mountNode, which is just the DOM node for the example.

Besides passing in fields, you can also pass in the value.

// Create some fields.
var fields = [
  {
    type: 'single-line-string',
    key: 'firstName',
    label: 'First Name'
  },
  {
    type: 'single-line-string',
    key: 'lastName',
    label: 'Last Name'
  }
];

// Render the form.
React.render(
  <Formatic fields={fields} value={{firstName: 'Joe', lastName: 'Foo'}}/>
, mountNode);
